# Frontend-Only Project

This workspace is configured as **frontend-only**.

## Active Project

- `zombie-city-escape/` (HTML + CSS + Vanilla JavaScript Canvas game)

No backend, database, API, or server code is required.

## Run Locally

Open directly:

- `zombie-city-escape/index.html`

Or run a static server:

```bash
cd zombie-city-escape
python -m http.server 8080
```

Then open `http://localhost:8080`.

## Deploy

Deploy `zombie-city-escape/` to any static host:

- Netlify
- Vercel (static)
- GitHub Pages
- Cloudflare Pages
