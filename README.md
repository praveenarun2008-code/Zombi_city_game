# Zombie City Escape

Zombie City Escape is a survival runner game where you keep moving, avoid deadly obstacles, and try to stay alive long enough to complete each level.

This game is made to feel intense. It is not supposed to be easy. The speed, obstacles, bosses, and pressure are all there to make every run feel dangerous.

## How To Play

Your goal is simple:

- Run as far as you can
- Stay alive
- Reach the target distance for the level
- Move on to the next level

If your health drops to zero, the run is over.

## Controls

- `Arrow Up` = Jump
- `Arrow Down` = Slide
- `P` = Pause
- `R` = Restart after losing
- `N` = Go to the next level after winning

## What You Need To Avoid

While running through the city, you will face:

- Broken cars
- Hanging signs
- Fire pits
- Boss zombies

You need to react fast because each obstacle needs good timing. A bad jump or a late slide can end the run quickly.

## What Helps You Survive

- Coins give you score and some health back
- Shield powerups protect you for a short time
- Slow-motion powerups make the run easier for a few seconds

These can save you, but only if you grab them at the right time.

## Why The Game Is Hard

This game gets harder on purpose.

- The speed increases as you survive longer
- Obstacles keep coming and can appear with little breathing room
- Boss zombies add extra pressure during the run
- You need both timing and focus, not just luck
- Each level asks you to survive longer than the last one

The idea behind the game is to make the player feel chased, stressed, and always one mistake away from losing.

## Challenge Level

I made this game to be a real challenge, not a calm or easy runner.

- Early levels help you understand the controls
- Mid levels test your consistency
- Later levels are meant to feel tough, fast, and unforgiving

If you win, it should feel earned.

## Best Way To Play

Start from the menu and choose a level:

- `zombie-city-escape/menu.html`

Try to beat your own score, learn the obstacle timing, and keep pushing farther each run.
